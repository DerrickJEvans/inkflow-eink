🔗 Connection Guide

Configure your Arduino UNO R4 or Seeed Studio XIAO ePaper display by pointing the C++ client to:

http://[server-ip]:5000/api/display/raw?device=xiao_screen&width=800&height=480

Microcontrollers download pre-dithered raw bytes directly over WiFi into screen SRAM for quick, low-power renders.

Configure the Python client on a Pi Zero by pointing the connection string in config to:

http://[server-ip]:5000/api/display/image.png?device=pi_screen&width=800&height=480

Pulls full dithered PNGs via standard HTTP GET. Supports Waveshare, Pimoroni Inky, and local mock previews.
